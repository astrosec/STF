説明

イテレータの抽象インターフェース．

STFの内部オブジェクト用に用意されるイテレータ群の共通インターフェース． すべて片方向のinput iteratorで，かつ返す値はdouble型に限定されている． 使い方は以下のとおり． 

        Object* obj = new Object();//走査対象
        Iterator it(obj);
        while(!it.end()){
                std::cout << (*it)() << ", ";
                ++it;
        }
