説明

コマンドの処理を行う機能マネージャ基底クラス．

ユーザーはこのクラスを書き換えることが許可されない． 機能マネージャの振る舞いをカスタマイズしたい場合，派生クラス側で仮想関数を実装する（Generation-Gapパターン）．
