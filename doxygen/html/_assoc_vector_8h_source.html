00074 // class template AssocVector
00075 // An associative vector built as a syntactic drop-in replacement for std::map
00076 // BEWARE: AssocVector doesn't respect all map's guarantees, the most important
00077 //     being:
00078 // * iterators are invalidated by insert and erase operations
00079 // * the complexity of insert/erase is O(N) not O(log N)
00080 // * value_type is std::pair<K, V> not std::pair<const K, V>
00081 // * iterators are random
